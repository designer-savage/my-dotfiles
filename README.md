# Hyprland Setup

Моя конфигурация Arch Linux + Hyprland + Quickshell

## Система

- **OS**: Arch Linux (kernel 6.19.9)
- **WM**: Hyprland 0.54.2
- **Shell**: Quickshell 0.2.1 (QML-based)
- **Terminal**: Kitty 0.46.2
- **Laptop**: Lenovo ThinkBook 15 G3 ACL
- **CPU**: AMD Ryzen 7 5700U @ 4.37GHz (16 threads)
- **RAM**: ~13.5 GB

## Компоненты

- **hyprland** — тайловый композитор
- **quickshell** — статус-бар на QML
- **kitty** — терминал
- **waybar** — альтернативный бар (резервный)
- **hyprlock** — экран блокировки
- **hypridle** — управление idle
- **hyprpaper** — обои
- **hyprpicker** — пипетка цветов
- **hyprshot** — скриншоты

## Структура

```
.config/
├── hypr/           # Конфиги Hyprland (модульные)
├── quickshell/     # QML-конфиг для статус-бара
├── kitty/          # Конфиг терминала
└── waybar/         # Альтернативный бар
```
